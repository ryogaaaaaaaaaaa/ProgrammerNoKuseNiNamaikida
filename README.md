# 記憶庭園の番人

Godot 4.6 で作った、逆ローグライク・デッキ構築ゲームです。  
プレイヤーは「眠る巨大樹の記憶庭園」を守る番人となり、上層から最深部へ降りてくる調査隊を、根道・胞子・結晶・霧・芽吹きの種子札で迎撃します。

## 遊び方

- カードをクリックして選び、盤面のタイルをクリックして使います。
- 土タイルには「根道を開く」、通路タイルには胞子灯や絡み根などを配置できます。
- 「迎撃開始」で複数拍がテンポよく進み、庭園の仕掛けと調査隊が連続で動きます。
- 調査隊には「直進」「外周寄り」「未探索優先」「罠回避」の探索ロジックがあります。
- 各日をしのぐと、3つの報酬から1つを選んでデッキを強化できます。
- 7日目を守り切ると勝利です。

## 操作

- 左クリック: カード選択、カード使用、ボタン操作
- 右クリック / Esc: 選択解除
- Space: 迎撃開始 / 迎撃中の早送り
- F1: 遊び方
- M: ミュート切り替え

## 起動

```bash
godot --path .
```

Godot エディタから開く場合は、このフォルダをプロジェクトとしてインポートしてください。

## アセット

- `assets/title_memory_garden.png`: Imagegen スキルで生成したタイトル背景です。
- 盤面、カード、ユニット、エフェクトは Godot の描画 API で生成しています。
